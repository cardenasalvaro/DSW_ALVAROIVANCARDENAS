git y github

objetivo general

diseñar un OVA para el curso de diseños de sitios web donde se profundice sobre la tematica de git y github

objetivos especificos

  • guardar el historial de cambios sobre el codigo de un proyecto
  • permitir a varias personas contribuir simultaneamente sobre un mismo proyecto
  • almacenar el historial del codigo localmente en la carpeta oculta .git dentro de la raiz de un proyecto
  • hacer commit repositorio local, esos cambios no se ven reflejados automaticamente en el repositor remoto
